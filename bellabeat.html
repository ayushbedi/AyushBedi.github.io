Bellabeat Data Analysis Case Study

Project Overview

As part of a real-world case study, I took on the role of a Junior Data Analyst at Bellabeat, a wellness technology company focused on women's health. The goal of this project was to explore smart device usage data to uncover behavioral trends and provide actionable insights that could help Bellabeat refine its marketing strategy and improve user engagement.

Using a public dataset from Fitbit (non-Bellabeat users), I analyzed various dimensions of wellness, including physical activity, caloric expenditure, sleep quality, and device engagement patterns. This analysis allowed me to draw key conclusions about user habits and generate strategic recommendations for Bellabeat's app and wearable products like Leaf, Time, and Spring.

Objectives

  • Analyze usage trends in smart device data across activity, calories, heart rate, sleep, and weight
  • Understand user behavior by segmenting active vs. inactive users and identifying peak performance times
  • Assess engagement gaps such as low step count days, short sleep durations, and sedentary time
  • Generate strategic marketing recommendations to improve user retention and wellness outcomes
  • Apply SQL and data visualization tools to extract insights, build dashboards, and communicate findings

Tools Used

SQL (BigQuery, DBeaver) Data Visualization (Tableau) Tableau Google Sheets

Data Cleaning and Validation

    ActivityDateTime DATETIME;

    SET ActivityDateTime = STR_TO_DATE(ActivityHour, '%m/%d/%Y %l:%i:%s %p');  
            
-- Check for NULLS  
SELECT *  
FROM hourlycalories_merged hm  
WHERE Id IS NULL OR ActivityHour IS NULL OR  
    Calories IS NULL;  
            
-- Count duplicate rows  
SELECT Id, ActivityHour, COUNT(*)  
FROM hourlycalories_merged hm  
GROUP BY Id, ActivityHour  
HAVING COUNT(*) > 1;

-- This query groups average calorie burn by custom parts of the day:
-- Early Morning (5 AM–11 AM), Afternoon (12 PM–5 PM), Evening/Night (6 PM–12 AM)

  CASE

Minute Sleep File Analysis

Sleep Stage Distribution

-- Unique Users
SELECT COUNT(DISTINCT Id) FROM minuteSleep_merged;
            
-- Average Session Length (min)
SELECT AVG(session_length) 
FROM (
    SELECT COUNT(*) AS session_length 
    FROM minuteSleep_merged 
    GROUP BY logId
    );
            
-- Sleep Stage Distribution (minutes)
SELECT 
    CASE value
        WHEN 1 THEN 'asleep'
        WHEN 2 THEN 'restless'
        WHEN 3 THEN 'awake'
    END AS sleep_stage,
    COUNT(*) AS total_minutes
FROM minuteSleep_merged
GROUP BY sleep_stage;
            
-- Average Bedtime & Wake Time (hour)
SELECT 
    AVG(HOUR(start_time) + MINUTE(start_time)/60.0) AS avg_bedtime,
    AVG(HOUR(end_time) + MINUTE(end_time)/60.0) AS avg_wake
FROM (
    SELECT 
        MIN(STR_TO_DATE(date, '%m/%d/%Y %r')) AS start_time,
        MAX(STR_TO_DATE(date, '%m/%d/%Y %r')) AS end_time
    FROM minuteSleep_merged
    GROUP BY logId
    );

Key Findings

Activity Patterns

Only 27.8% of recorded days meet the recommended 10,000 step goal. Activity levels are approximately 15% lower on weekends compared to weekdays, with Tuesday and Wednesday showing peak activity.

Most users (72%) fall below physical activity guidelines, presenting an opportunity for targeted motivation features.

Sleep Insights

Users average 6.8 hours of sleep, below the recommended 7-9 hours. More active users tend to have better sleep quality, with a positive correlation between steps and total sleep time.

Users spend an average of 39 minutes in bed before falling asleep, indicating potential for sleep hygiene features.

Usage Patterns

Device usage drops by approximately 40% after the first month, suggesting engagement challenges. User activity varies significantly, with top performers achieving more than twice the average step count.

Sedentary time averages nearly 17 hours daily, highlighting an opportunity for movement reminders.

Additional Data to Expand the Analysis

While the Fitbit dataset provides valuable behavioral insights, incorporating additional data sources would allow for a deeper and more personalized analysis. These enhancements could help Bellabeat better understand user needs and tailor product offerings:

  • Demographics: Age, occupation, and location would help in segmenting users and designing targeted features.
  • Health & Lifestyle Goals: Knowing if a user is aiming for weight loss, better sleep, or stress reduction could guide personalized recommendations.
  • Nutrition/Food Intake: Correlating calorie intake with activity and sleep would create a holistic view of user wellness.
  • Qualitative Feedback: User satisfaction scores or app reviews could help explain behavioral patterns and highlight areas for improvement.
  • Bellabeat-Specific Usage Data: Comparing this dataset with Bellabeat users would validate assumptions and highlight unique user behaviors.

How Could the Team Apply These Insights?

The insights uncovered from the analysis can be translated into impactful strategies across product development, marketing, and user engagement efforts. Here's how different teams at Bellabeat could take action:

  • Product Development: Use afternoon activity peaks and sleep patterns to trigger smarter push notifications and personalized nudges.
  • UX Design: Add features like micro-movement reminders, bedtime alerts, and user streaks to promote healthy habits.
  • Marketing Strategy: Highlight the link between activity and improved sleep in campaigns, tailored especially for women's wellness goals.
  • User Retention: Create tiered reward systems and personalized progress dashboards to keep users motivated over time.
  • Customer Segmentation: Identify high vs. low activity users and deliver targeted challenges, reminders, or tips based on behavior.

Strategic Recommendations

  1. Boost Activity Engagement Through Afternoon Challenges: Analysis shows user activity peaks in the afternoon (12 PM–5 PM). Leverage this by introducing mid-day challenges or reminders like "3 PM Walk Boost" to turn natural activity spikes into consistent habits.
  2. Sleep Optimization and Restless Time Coaching: With an average of just under 6 hours of sleep and 15% spent restless or awake, Bellabeat can introduce sleep coaching features. These may include bedtime reminders, relaxation audio, or a "Sleep Score" based on restfulness and duration.
  3. Segment and Reward High Performers to Drive Motivation: A small group of users consistently achieve high steps and activity minutes. Bellabeat can motivate all users by introducing a tiered recognition system — such as "Top Movers" or "Weekly Wellness Stars" — encouraging both consistency and improvement.
  4. Combat Sedentary Behavior with Micro-Movement Goals: Users are sedentary for nearly 17 hours daily. Bellabeat can introduce hourly "stand and stretch" prompts and track streaks to gently reduce sedentary time. Framing these as wellness wins helps drive positive behavior change.

These recommendations are grounded in user behavior patterns from the Fitbit dataset and align with Bellabeat's mission to promote holistic wellness through technology. Implementing them could improve product value, user satisfaction, and long-term retention.

Final Conclusion:

The Bellabeat analysis reveals a clear gap between user behavior and recommended health standards. Across the dataset, only 27.8% of logged days meet the 10,000-step activity goal, and the average sleep session is just under 6 hours, falling short of the recommended 7–9 hours. These findings indicate that while users are engaging with their devices, they are not consistently achieving healthy lifestyle benchmarks.

User activity is concentrated in the afternoon and evening, suggesting that interventions or nudges during these peak windows could be effective. Interestingly, users with higher physical activity levels also show better sleep quality, reinforcing the idea that wellness habits are interconnected and can be improved through holistic design strategies.

The data also points to a significant drop in device engagement after the first month, signaling a key challenge in user retention. Most users exhibit high sedentary time—averaging nearly 17 hours daily—which presents a compelling opportunity for Bellabeat to develop movement-based reminders and motivational nudges.

In short, there's strong evidence that Bellabeat can play a greater role in helping users build sustainable, healthy habits through personalized guidance, better sleep support, and behavior-based engagement strategies.
