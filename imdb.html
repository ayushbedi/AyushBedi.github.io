IMDb Movie Dataset Analysis

Project Overview

This project explores the IMDb dataset, focusing on movies released between 1894 and 2025. With a total of 135,554 movies and an average user rating of 6.04, the analysis aims to uncover trends in movie quality, viewer preferences, and popularity.

The merged dataset combines movie metadata (such as title, year, runtime, and genre) with user ratings and vote counts. This allows for a multifaceted analysis of cinematic trends across time, genre, and viewer engagement. The most frequently represented genres include Drama, Comedy, Documentary.

Objectives

  • Analyze trends in movie ratings across different decades
  • Identify genres with consistently high or low viewer ratings
  • Explore correlations between number of votes and average rating
  • Determine the most popular movies by rating and vote count
  • Segment movies by genres to study viewing preferences

Tools Used

SQL Pandas Matplotlib / Seaborn Jupyter Notebook

IMDb Data Cleaning Process Using Microsoft Excel

Data Acquisition and Initial Filtering

Downloaded raw datasets (name.basics, title.basics, and title.ratings) from IMDb's official datasets available at datasets.imdbws.com each containing 1,048,576 rows of data.

Data Structure

Initially, the dataset (Title_basics_movies) consisted of the following columns:

  • tconst (Title Identifier)
  • titleType (Type of Media)
  • primaryTitle and originalTitle
  • isAdult (Adult Content Indicator)
  • startYear and endYear
  • runtimeMinutes
  • genres

Data Cleaning and Filtering

  • Filtered the extensive title.basics dataset specifically to focus on movie entries, narrowing it down to 229,025 relevant entries.
  • Filtered out entries lacking runtimeMinutes information to maintain consistency and accuracy in duration-based analyses.
  • Cleaned the dataset by removing invalid or missing year (/N) entries from the startYear column, ensuring accurate temporal analyses.
  • Removed the unnecessary endYear column, as movies typically do not extend across multiple years.
  • Excluded movies without genres classifications to facilitate clear and meaningful genre-based insights. This resulted in a concise, analysis-ready dataset comprising 161,213 rated movie entries.

These rigorous steps ensured a clean, structured, and enriched dataset, facilitating comprehensive analysis of cinematic trends and viewer preferences.

Movie Counts by Year Range (1890–Present)

--Movie Counts by Year Range (1890–Present)
SELECT
  CASE
    WHEN startYear BETWEEN 1890 AND 1910 THEN '1890-1910'
    WHEN startYear BETWEEN 1911 AND 1930 THEN '1911-1930'
    WHEN startYear BETWEEN 1931 AND 1950 THEN '1931-1950'
    WHEN startYear BETWEEN 1951 AND 1970 THEN '1951-1970'
    WHEN startYear BETWEEN 1971 AND 1990 THEN '1971-1990'
    WHEN startYear BETWEEN 1991 AND 2010 THEN '1991-2010'
    WHEN startYear >= 2011 THEN '2011+'
    ELSE 'Unknown'
  END AS year_range,
  COUNT(*) AS movie_count
FROM `tough-shard-442823-m5.IMDb_Dataset.movies`
GROUP BY year_range
ORDER BY year_range;
